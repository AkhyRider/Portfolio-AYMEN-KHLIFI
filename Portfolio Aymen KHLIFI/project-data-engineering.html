def clean_dataset(df):
    # Traitement des valeurs manquantes
    df['age'] = df['age'].fillna(df['age'].median())
    
    # Détection des outliers (méthode IQR)
    Q1 = df['score'].quantile(0.25)
    Q3 = df['score'].quantile(0.75)
    IQR = Q3 - Q1
    df = df[~((df['score'] < (Q1 - 1.5 * IQR)))]
    
    # Feature Engineering
    df['engagement_rate'] = df['clicks'] / df['impressions']
    return df

Défis du projet

Ce projet portait sur la manipulation de jeux de données massifs (plusieurs millions de lignes) issus de logs serveurs. La problématique majeure consistait à transformer des données brutes "sales" en variables prêtes pour l'entraînement de modèles de Machine Learning.

Solution Implémentée

J'ai mis en place des pipelines automatisés utilisant Pandas et NumPy. L'accent a été mis sur la robustesse du code pour gérer les valeurs nulles imprévues et les types de données incohérents.

Apports Techniques

  • Réduction du temps de nettoyage de 70% grâce à la vectorisation.
  • Création de 12 nouvelles features prédictives basées sur des corrélations statistiques.
  • Normalisation des données pour assurer l'équité des modèles de clustering.
