Data & Gamification

Jeu Foot Locker — Collecte de Données

Python JavaScript Data Collection
Contexte

Foot Locker souhaitait engager sa communauté jeune à travers une expérience ludique tout en récupérant des données qualifiées sur les préférences des utilisateurs. L'objectif était double : branding et enrichissement CRM.

Rôle

Développeur et Data Analyst. J'étais responsable de la conception technique du mini-jeu et de l'architecture de collecte des données utilisateurs en arrière-plan.

Processus
  • Développement de l'interface jeu (HTML/JS).
  • Création de scripts Python pour structurer les données brutes (scores, temps de jeu, choix).
  • Mise en place de règles de confidentialité et nettoyage des entrées.
Résultats
  • Engagement utilisateur supérieur à 3 minutes/session.
  • Base de données enrichie de +500 profils qualifiés.
  • Tableau de bord des préférences produits généré pour l'équipe marketing.
Data Engineering

Nettoyage & Feature Engineering

Pandas Python

Contexte : Dataset brut contenant des incohérences majeures empêchant toute analyse fiable.

Intervention : Développement de pipelines de nettoyage automatisés sous Pandas. Traitement des valeurs nulles, détection d'outliers et normalisation. Création de nouveaux indicateurs (KPIs) pour évaluer la fiabilité technique des entrées.

Modélisation

Score de Qualité Pondéré

Scikit-learn Statistiques

Objectif : Créer une métrique unique pour classer la performance de contenus multimédias.

Solution : Algorithme pondérant la Popularité (60%) et la Stabilité de l'audience (40%). Ce score permet désormais de prioriser les contenus à forte rétention pour les campagnes publicitaires.
