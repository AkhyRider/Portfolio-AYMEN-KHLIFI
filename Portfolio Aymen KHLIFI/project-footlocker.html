À propos du projet

Ce projet a été réalisé dans le but de moderniser l'acquisition de données pour Foot Locker. Au lieu d'un formulaire classique, nous avons opté pour une approche de "gamification".

Les utilisateurs participent à un mini-jeu interactif inspiré de l'univers du basketball. En fin de session, des récompenses sont offertes en échange d'informations de profilage.

Approche Technique

L'interface a été développée en JavaScript pur pour garantir une performance optimale sur mobile. En coulisses, un script Python traite les flux de données pour identifier les segments de clients les plus engagés.

Résultats Clés

  • Augmentation du taux de complétion des profils de 45% par rapport aux méthodes traditionnelles.
  • Collecte de données comportementales inédites (temps de réaction, préférences visuelles directes).
  • Mise en place d'un dashboard marketing en temps réel.
